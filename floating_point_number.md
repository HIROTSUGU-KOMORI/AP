
# _数値表現_

---
## _基数変換_

---
### _浮動小数点数_

- コンピュータにおける数値を表現する方法の一つ。
- 小数点を含む数値を利用するときに使用される。
- 32ビット形式の浮動小数点数の場合、符号1ビット(正の数0、負の数1)、指数部7ビット、仮数部24ビット
- 最上位桁を0以外の数字にする(正規化)

```
10進数の「0.01」を浮動小数点数で表現する。
10進数 : 0.01
    ↓　変換
2進数 : 0.000000101000111101011100001010・・・
    ↓　小数点の位置を調整し、最上位桁を0以外にする(正規化)
正規化後 : 0.1010001111010111001010 * 2^-6
∴符号0、指数部1111010、仮数部1010001111010111001010
------------------------------------------------------------
ちなみに、負数の「-6」は"2の補数"で表現する。
110(6)を32ビット形式に合わせて、0000110
すべてのビットを反転し1加える : 1111010(-6)
------------------------------------------------------------
```
- 「IEEE754」の浮動小数点数は、符号1ビット、指数部8ビット、仮数部23ビット


```
10進数の「0.01」を「IEEE754」形式で浮動小数点数で表現する。
10進数 : 0.01
    ↓　変換
2進数 : 0.000000101000111101011100001010・・・
    ↓　小数点の位置を調整し、最上位桁を「1.***」の形にする(正規化)
正規化後 : 1.010001111010111001010 * 2^-7
    ↓　さらに、バイアスを加算する(ゲタをはかせ、負の数も正の数で表現する)
加算後 : -7 + 127 = 120
    ↓　2進数に変換
変換後 : 01111000
∴符号0、指数部01111000、仮数部010001111010111001010
```
